## Project setup

```shell
yarn
```

### Compiles and hot-reloads for development

```shell
yarn dev
```

### Create menu tree Sql

```shell
yarn run menu
```

### Create api or page

```shell
npm run new
```

### Compiles and minifies for production

```shell
yarn build
```

### Run your tests

```shell
yarn test
```

### Lints and fixes files

```shell
yarn lint
```
